Installation Guide

Option 1: Pre-Configured Installation (Recommended)
  1. Download the optimized .ZIP package for your specific game launcher platform (Steam/Itch or MS/Epic/Xbox).
  2. Extract the contents of the password-protected .ZIP file directly into your main Among Us game root installation directory.
  3. Launch the game normally. The mod menu will handle hook configurations and load automatically on startup.
  4. Click the Delete key on your physical keyboard to toggle open the overlay menu interface.
Option 2: Standalone DLL Injection
  1. Download the standalone FabMenu.dll asset file.
  2. Ensure you already have the standard BepInEx Injector toolkit correctly set up inside your local game files directory.
  3. Place the FabMenu.dll file into your game's BepInEx plugins folder directory paths (Among Us/BepInEx/plugins).
  4. Launch the game through your launcher to let the runtime loader initialize the mod menu modules.
  5. Click the Delete key on your physical keyboard to toggle open the overlay menu interface.
